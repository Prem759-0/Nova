## ✨ Why NOVA?

NOVA is a frontend‑only demo of a universal command centre that merges AI with stunning UI.  
It feels like **Apple’s design + Linear’s focus + Raycast’s speed** — wrapped in a cosmic, glass‑morphism aesthetic.

- **360° interaction** — cursor‑responsive 3D orb, magnetic buttons, glass cards that lift on hover  
- **Command palette** — press `⌘K` to summon a fully animated overlay with perfect spring physics  
- **Cinematic scroll** — each section reveals with motion and blur  
- **90fps smooth** — GPU‑accelerated, best practices
